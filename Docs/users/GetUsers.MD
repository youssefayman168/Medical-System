# Get Users

---

# This Endpoint Allows Retrieving All Users' Information 

# Endpoint: https://apis.kah-moh.com/users/get-user/{userID}/

> NOTE: replace `userID` with the desired user's ID

# METHOD: GET

# Response Example:

```

{
    "data": {
        "username": "YOUSSEF",
        "email": "yosofaymanessawy@gmail.com",
        "job_title": "JOB",
        "national_id": 15154
    }
}

```
